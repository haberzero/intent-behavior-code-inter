# IBC-Inter 类型与运算符详细说明

## 类型总览

### 基本类型

- int
- float
- str
- None
- callable：通用可调用对象（函数、LLM 函数、行为 Lambda）

### 容器类型

- list
- dict：key 仅允许 int 或 str，value 允许任意类型

## 基本类型说明

### int

```ibc-inter
int 数量 = 42
int 结果 = 数量 * 2
```

### float

```ibc-inter
float 税率 = 0.05
float 总计 = 100.0 * (1 + 税率)
```

### str

```ibc-inter
str 名字 = "张三"
str 问候 = "你好，" + 名字
```

### None

```ibc-inter
str 为空 = None
```

### callable

`callable` 类型用于存储可被调用的对象。它既可以承接普通函数，也可以承接被 Lambda 化的行为描述行。

```ibc-inter
# 承接行为描述行
callable 分析 = @~ 分析 $text 的主要意图 ~

# 承接普通函数
func add(int a, int b) -> int:
    返回 a + b
callable 运算 = add
```

## 运算符说明

### 算术运算（int/float）

```ibc-inter
int a = 10
int b = 3
int 加法 = a + b
float 除法 = a / b
int 取模 = a % b
```

### 比较运算

1. **数值与字符串比较**：支持 `==`, `!=`, `<`, `>`, `<=`, `>=`。
2. **None 比较**：`None` 允许与**任意类型**进行 `==` 和 `!=` 比较。如果比较双方有一个为 `None` 且另一个不是，则 `==` 为假，`!=` 为真。

```ibc-inter
if a == None:
    pass
if "text" != None:
    pass
```

### 逻辑运算

```ibc-inter
if (a > b and b > 0):
    执行语句
if (a > b or b > 0):
    执行语句
if not (a > b):
    执行语句
```

### 位运算（int）

```ibc-inter
int 位与 = a & b
int 位或 = a | b
int 位异或 = a ^ b
int 位非 = ~a
int 左移 = a << 2
int 右移 = a >> 1
```

### 行为描述（Behavior）

行为描述是一种特殊的表达式，使用 `@~...~` 包裹。支持在内容中使用 `$变量` 或 `$__表达式__` 进行动态插值。

**插值规则：**
1. **简单插值**：`$变量名`，直接将变量值转化为字符串。
2. **复杂插值**：`$__表达式__`（双下划线包裹），支持任意合法的 IBC-Inter 表达式，如 `$__user.name__` 或 `$__items[0].price__`。

```ibc-inter
str 结果 = @~分析数据 $data.id~
for @~等待条件满足 $__count > 10__~:
    pass
```

## LLM 返回值解析 (Parsing)

当 LLM 函数定义了非 `str` 的返回类型，或处于分支/循环判定场景时，解释器会自动进行鲁棒性解析。**开发者始终可以通过调试器查看解析和映射的具体过程。**

| 目标类型 | 解析策略 | 默认识别映射 |
| :--- | :--- | :--- |
| **int / float** | 提取文本中出现的第一个数字序列。 | - |
| **bool** | 识别并映射逻辑关键词。 | `1`, `true`, `yes`, `ok` -> **True**<br>`0`, `false`, `no`, `fail` -> **False** |
| **list / dict** | 提取第一个 `[...]` 或 `{...}` 块，自动剥离 Markdown 代码块。 | - |

> **提示**：上述解析逻辑仅在赋值给显式类型或处于控制流判定时生效。对于 `var` 类型或直接 `print(@~...~)`，将保留原始字符串输出。

```ibc-inter

```

## 运算符优先级

运算符优先级从高到低：

1.  **括号与访问**：`()` (括号), `.` (属性访问), `()` (调用), `[]` (索引)
2.  **一元运算**：`+x`, `-x`, `not`, `~x` (位非)
3.  **算术（乘除）**：`*`, `/`, `%`
4.  **算术（加减）**：`+`, `-`
5.  **位移运算**：`<<`, `>>`
6.  **比较运算**：`<`, `<=`, `>`, `>=`, `==`, `!=`
7.  **位与**：`&`
8.  **位异或**：`^`
9.  **位或**：`|`
10. **逻辑与**：`and`
11. **逻辑或**：`or`
12. **赋值**：`=`, `+=`, `-=`, `*=`, `/=`, `%=`
13. **行为描述**：`@~...~` (作为独立表达式解析)

> **开发者注**：IBC-Inter 严格遵循上述优先级。当逻辑复杂时，**强烈建议使用括号 `()` 显式包裹**，以确保代码意图的可读性与控制精度。
