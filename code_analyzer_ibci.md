根据对 `core/compiler` 和 `core/runtime` 代码的深入分析，IBCI (Intent-Behavior Code Inter) 的支持语法和内置模块总结如下：

### **1. 核心语法 (Control Flow)**
IBCI 提供了完整的命令式编程控制流，并针对 AI 场景进行了扩展。

- **基础控制流**：
  - `if`, `elif`, `else`：标准条件分支。
  - `while`：支持 `while condition if filter:` 形式的带过滤循环。
  - `for`：支持 `for item in collection`。
  - `for condition:`：条件驱动模式（类似于 `while` 但统一在 `for` 语法下）。
  - `break`, `continue`, `pass`, `return`：标准跳转指令。
- **异常处理**：
  - `try`, `except`, `else`, `finally`：标准错误捕获机制。
  - `raise`：主动抛出异常。
  - `retry`：重新尝试当前代码块。
- **AI 弹性语法 (IES 2.1+)**：
  - `llmexcept`：专门用于捕获 AI 模型生成失败或不符合预期时的异常块。
  - `llmretry`：AI 专用的声明式重试语法。

### **2. AI 特色语法 (AI Blocks & Intent)**
IBCI 的核心在于将"意图"与"行为"显式建模。

- **意图 (Intent)**：
  - `intent "content": block`：定义一个带有特定意图的作用域。
  - `@mode "content" statement`：意图简写，作用于随后的单条语句。
  - **意图模式 (Modes)**：
    - `+` 或 `append`/`add`：追加意图。
    - `!` 或 `override`/`exclusive`：覆盖当前意图栈。
    - `-` 或 `remove`/`delete`：移除特定意图。
  - **标签 (Tags)**：支持 `#tag` 语法对意图进行分类。
- **行为 (Behavior)**：
  - `~...~`：**延迟行为**。创建一个 `IbBehavior` 对象，存储提示词模板，但不立即执行。
  - `@~...~`：**即时行为**。立即调用 LLM 执行该描述并返回结果。
  - **插值 (Interpolation)**：在行为块中使用 `$var` 或 `$(expr)` 进行变量注入。

### **3. 类型系统 (Types)**
IBCI 采用"一切皆对象"的模型，并支持静态类型标注与动态类型转换。

- **基础类型**：`int`, `float`, `str`, `bool`, `None`。
- **容器类型**：`list` (`[...]`), `dict` (`{key: value}`)。
- **类型标注**：支持在变量定义和循环中指定类型，如 `for int i in range(10)`。
- **类型转换 (Casting)**：支持 `(Type) expression` 语法的显式强制转换。
- **特殊类型**：
  - `behavior`：代表延迟执行的 AI 逻辑。
  - `callable`：可调用对象（函数或方法）。
  - `void`, `Any`：用于类型系统的占位符。

### **4. 内置模块与原语 (Intrinsics)**
运行时通过 `IntrinsicManager` 注入了一系列核心函数和模块。

- **`io` 模块**：
  - `print(*args)`：输出到宿主环境，遵循 `__to_prompt__` 协议。
  - `input(prompt)`：从宿主获取输入。
- **`collection` 模块**：
  - `len(obj)`：获取容器长度或调用对象的 `len` 协议。
  - `range(start, stop, step)`：生成整数序列。
- **`meta` 模块**：
  - `get_self_source()`：获取当前执行模块的源代码，用于自省。
- **`ai` 模块**：
  - 提供 `auto_intent_injection` 等配置，控制 LLM 执行时的意图注入策略。
- **内置协议方法**：
  - `__add__`, `__sub__`, `__eq__` 等：支持标准的算术与比较运算符重载。
  - `__to_prompt__`：定义对象如何被转化为 AI 可理解的提示词字符串。

### **5. 运算符支持**
- **算术**：`+`, `-`, `*`, `/`, `//`, `%`, `**`
- **位运算**：`&`, `|`, `^`, `<<`, `>>`, `~`
- **逻辑**：`and`, `or`, `not`
- **比较**：`==`, `!=`, `<`, `<=`, `>`, `>=`（支持链式比较，如 `a < b < c`）
- **引用**：`$var`（在意图/行为块中引用上下文变量）
